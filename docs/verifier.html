Datos protegidos

Confidencialidad: el verificador no descarga evidencia sensible sin permiso.
Minimizacion: solo se muestran datos necesarios para acreditar autenticidad.
Acceso reforzado: consultas sensibles pueden exigir credencial institucional o doble factor.
Registro: toda consulta queda asociada al expediente como evento de trazabilidad.
